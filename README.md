# data_analytics_capstone_project

## About the project
This project was completed as part of a Riipen experiential learning engagement with an analytics consulting firm. Our team was tasked with designing and delivering an interactive dashboard solution to help account managers assess client performance, profitability, and risk across their portfolio. The goal was to surface high-level, strategic, and actionable insights in a clear, accessible format to grow the business more effectively.

## My role
**Analyst & Communications Lead**
- Led stakeholder communication between the student team and the client throughout the engagement
- Translated client requirements into analytical objectives
- Contributed to, and formatted, written reports summarizing findings, methodology, and strategic recommendations
- Contributed to dashboard design decisions, focusing on layout clarity and account manager usability

- Designed a BI dashboard enabling account managers to assess client profitability, risk exposure, and performance trends in a single view
